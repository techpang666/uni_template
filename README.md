
# **基于uViewUI的uniapp模板项目**
>基于程序员的惰性需求配置的uniapp模板项目🤪  

------
## **食用教程**
- clone项目
- 撤掉`.git`文件夹进行自己的版本控制
- 通过HX编辑器打开并配置小程序App🆔
- 通过微信开发者工具运行项目即可

------
## **一些内置能力**
- [X] 基于promise封装的请求库
- [ ] 全局css初始化
- [ ] 时间格式化工具
- [ ] 更多能力敬请期待🚀
