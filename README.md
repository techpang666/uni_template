
# **基于uViewUI的uniapp模板项目**
>基于程序员的惰性需求配置的uniapp模板项目🤪  

------
## **食用教程**
- clone项目
- 撤掉`.git`文件夹进行自己的版本控制
- 通过HX编辑器打开并配置小程序App🆔
- 通过微信开发者工具运行项目即可

------
